Mein Wissen über Cloud

Derzeit absolvieren wir ein Cloud-Modul, in dem wir an einem Projekt mit AWS arbeiten. Mit Azure habe ich jedoch weniger Erfahrung.

Cloud Computing Collage

Cloud Computing ist eine Technologie, die es ermöglicht, Daten und Anwendungen über das Internet zu speichern und abzurufen.

  • Vorteile: Skalierbarkeit, Flexibilität, Kostenersparnis.
  • Technologien: Public Cloud, Private Cloud, Hybrid Cloud.
  • Anwendungen: Datenspeicherung, Software as a Service (SaaS), Plattform as a Service (PaaS).

Cloud Computing hat die Art und Weise verändert, wie Unternehmen IT-Ressourcen bereitstellen und nutzen.
